Completed Projects

Projects that are completed and/or in production. Many can only be shared
conceptually, while some will have links to GitHub repos or the output.

  • Production Data Warehouse Example

    An example of a basic production data warehouse using entity relationship diagrams (ERD) ,example queries, and example reporting that comes from the data warehouse.

  • Production Simulation and Loading

    Level loading multiple production lines can be incredibly difficult without appropriate visualization. Using scheduled and simulated demand can dramatically improve long lead time projects and on-time delivery.

  • Time Series Predictions

    Although there are many methods of simulating future needs, time series models provide an exciting way to make predictions without having to have huge volumes of multi-dimensional data.

  • COVID-19 Production Impact

    For many companies 2020 was a challenging year. This project analyzes the impact of COVID-19 on a period of 2020 and the direct correlations between COVID-19 cases/deaths and the on-time metrics used to determine company success.

  • Hadoop/Scala Experiments

    Some experimetns using Hadoop and Scala on a single node cluster inside AWS.

  • Direction of Trade

    An analysis of international trade using R.

  • Employee Turnover Prediction

    Attempting to predict highest flight risk employees and the most impactful features. Jupyter used for analysis and also contains a PowerPoint summation.

  • Production Time Series

    Using Python to create time series models to predict on-time completion for production metrics.

  • Kubernetes at Home

    A collection of yaml files used to deply a 2 node Kubernetes cluster at home. This example contains the setup to run Plex (media server), PiHole and dashboards. This can be extended for many other applications (and is in my real deployment).

  • Construction Load Estimator

    A Jupyter notebook for comparing a traditional loan to purchase a home and a construction loan (which includes all the closing and ongoing costs), as well as a comparison to renting outcomes.
